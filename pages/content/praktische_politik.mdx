# Wie kann ich mich überhaupt engagieren in der Schweiz?

Ein Lernpfad für praktische politische Aktivität.

## Themen

- Auf die Strasse gehen
- Eine Petition starten
- Eine Interessensgemeinschaft gründen
- Gleichgesinnte mobilisieren
- Die Medien für dein Anliegen gewinnen
- Das Referendum ergreifen
- Eine Initiative lancieren

## Tools

- Medienmitteilung verfassen (Deutsch)
