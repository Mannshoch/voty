# Curriculum

## Was ist Demokratie

### Staatsformen

- Staatsform, wie wird ein Land gesteuert
- Monarchie -> Mono = ein Herscher regiert alleine (König, Kaiser)
- Aristokratie -> Aris = die Adligen regieren
- Demokratie -> Demos = Das Volk regiert
- Topic: Demokratie-Index (https://de.wikipedia.org/wiki/Demokratieindex)

### Direkte vs repräsentative Demokratie

- Gewaltentrennung: Judikative, Exekutive, Legislative
- Direkte Demokratie: das Volk kann selber über wichtige Entscheide abstimmen
- Repräsentative Demokratie.: Das Volk wählt (beispielsweise alle vier Jahre) ein Parlament.
- Parlament, gewählte Abgeordnete

### Wichtige Grundvoraussetzungen für eine funktionierende Demokratie

- Menschenrechte
- Meinungsfreiheit
- Pressefreiheit

# Meta

- Moderne Modelle der Politikkompetenz postulieren vier Politikkompetenzdimensionen: Fachwissen, Urteilsfähigkeit, Handlungsfähigkeit, sowie Einstellung und Motivation (Detjen, Massing, Richter & Weisseno, 2012).
- Der Fokus auf der Dimension Einstellung und Motivation hat gute Gründe: bisherige Massnahmen im Bereich der politischen Bildung fokussieren sich laut einer Studie der Universität Bern fast ausschliesslich auf die Dimension Fachwissen, das Wecken von Interesse wird dabei ausgeklammert (Stadelmann-Steffen, Koller & Sulzer, 2015)
