# Als Lernender registrieren

Hey, schön bist Du hier! Wenn Du auf voty.ch ein neues Konto eröffnen möchtest,
brauchst Du dazu von Deiner Lehrperson einen Einladungslink. Der sieht in etwa so aus:

<pre> &nbsp; https://voty.ch/i/XYZeinlangercode</pre>

Falls Dich eine Lehrperson bereits mit Deiner Email-Adresse angemeldet hat,
dann klicke auf den Link im Bestätigungsemail, das Du erhalten hast.
